Kaspi Bulk Parser
Веб-парсер для Kaspi.kz, использующий Axios, Fuse.js и MongoDB.
Необходимые компоненты:

Node.js (рекомендуется версия 14 или выше)
MongoDB (работает локально или на Atlas)
npm (менеджер пакетов Node)


```bash
npm install
npm start


Отдельно:
npm run scrape (для парсинга)
npm run map (для маппинга)
npm run db (для загрузки в mongodb)
